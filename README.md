# Learn and Grow Initiative Website

## Project Overview
A responsive website for the Learn and Grow Initiative, a non-profit organization dedicated to empowering rural children with quality education and creative opportunities in Eastern Cape, South Africa.

## Project Structure
learn-and-grow-website/
index.html          Homepage
about.html          About Us page
services.html       Programmes page
enquiry.html        Enquiry form page
contact.html        Contact page with interactive map
css/
style.css       Main stylesheet
js/
script.js       JavaScript functionality

## Features Implemented

### Core Website Pages
- Homepage: Introduction and call-to-action sections
- About Us: Organization background, mission, vision, and team
- Programmes: Educational programmes with interactive cards
- Enquiry: Volunteer and sponsorship enquiry form
- Contact: Contact information with interactive map

### JavaScript Functionality
- Interactive Elements: Accordions, tabs, and modals
- Form Validation: Client-side validation with error handling
- Image Gallery: Lightbox functionality for programme images
- Search Feature: Content filtering for programmes
- Interactive Map: Leaflet.js integration for location display

### Design Features
- Responsive Design: Mobile-first approach
- Modern UI: Clean, professional appearance
- Accessibility: Semantic HTML and proper contrast ratios
- Performance: Optimized images and efficient code

### SEO Optimization
- Semantic HTML structure
- Meta tags and descriptions
- Proper heading hierarchy
- Image alt attributes
- Clean URL structure

## Technologies Used
- HTML5: Semantic markup
- CSS3: Modern styling with Flexbox/Grid
- JavaScript: ES6+ for interactivity
- Leaflet.js: Interactive maps
- Unsplash: High-quality images
